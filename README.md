# Инструкция по запуску

## Описание
Программа предназначена для фильтрации содержимого входных текстовых файлов и распределения данных по типам (целые числа, вещественные числа, строки) в отдельные выходные файлы.

Поддерживается:
- сбор краткой и полной статистики по данным;
- режим добавления данных в существующие файлы.

## Требования
- **Версия Java:** Java 21.0.2
- **Система сборки:** Gradle 8.5
- **Сторонние библиотеки:** 
  - picocli 4.7.7 — библиотека для обработки аргументов командной строки. Ссылка: https://picocli.info/
  ```kotlin 
  implementation("info.picocli:picocli:4.7.7")
  ```
- **Замечания:**
  - Входные файлы должны быть в кодировке UTF-8. Файлы в других кодировках (например, Windows-1251) могут вызвать ошибки чтения.
## Сборка и запуск программы
1. Склонируйте проект:
    ```bash
       git clone https://github.com/Juanoff/file-content-filter.git
       cd file-content-filter
    ```
2. Перейдите на стабильную ветку master:
    ```bash
    git checkout master
    ```
3. Перейдите в корневую директорию проекта, где находится файл **build.gradle.kts**. 
4. Выполните сборку с помощью Gradle Wrapper: 
    ```bash 
    ./gradlew shadowJar
    ```
5. После успешной сборки JAR-файл появится в директории **build/libs** с именем **file-content-filter-1.0.0.jar**. 
6. Запустите программу с помощью команды:
    ```bash
    java -jar build/libs/file-content-filter-1.0.0.jar [флаги] [входные_файлы]
    ```
    Пример: 
    ```bash
    java -jar build/libs/file-content-filter-1.0.0.jar -a -s -p sample- file1.txt file2.txt
    ```
